2.for循环和while循环的区别
1、循环条件不一样

2、死循环条件不一

3、循环次数不确定

　　使用区别：如果你想在循环结束后，继续使用控制条件的那个变量，用while循环，

　　　　　　　否则用for循环。不知道用for循环。
　　　　　　　因为变量及早的从内存中消失，可以提高内存的使用效率。

　　其实还有一种场景的理解:
　　　　如果是一个范围的，用for循环非常明确。
　　　　如果是不明确要做多少次，用while循环较为合适。
　　　　举例：吃葡萄。

优先考虑for，其次考虑while，最后考虑do...while
